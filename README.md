# golang_web_api

ジブリの映画を検索して、その映画に登場するキャラクターを表示します。

## 使い方

### 0. 準備

```zsh: クローン
# リポジトリをクローン
$ git clone https://github.com/nyantarou2001002/WebAPI.git
# ワークディレクトリに移動
$ cd golang_web_api
```

### 1. サーバー起動

```zsh: サーバー起動
$ go run backend/main.go
```

### 2. UIの表示

```zsh: UIの表示
$ open frontend/index.html
```
